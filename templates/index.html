Sistema de Detección de Neumonía

Planteamiento

La neumonía es una de las principales causas de mortalidad infantil a nivel global (OMS, 2023). Su diagnóstico temprano mediante rayos X es crítico, pero en regiones con escasez de radiólogos, la interpretación de imágenes puede retrasarse

Falsos negativos/positivos en diagnósticos manuales. Saturación de sistemas de salud en áreas rurales. Falta de estandarización en la evaluación de imágenes.

Desarrollo de un modelo de CNN (DenseNet) entrenado con el dataset Chest X-Ray para clasificar automáticamente imágenes de rayos X y detectar neumonía con alta precisión.

Justificación

Este proyecto busca desarrollar un modelo de machine learning para detectar neumonía en radiografías de tórax, cnetrandonos en la necesidad crítica en salud global, la demora en diagnósticos precisos, especialmente en regiones con escasez de especialistas. Utilizando un dataset público de Kaggle (Chest X-Ray Images), se entrenará un modelo de ML "CNN (DenseNet)" para clasificar imágenes, reduciendo falsos negativos que ponen vidas en riesgo, proponemos una aplicación web en flask que optimizará recursos médicos, disminuirá tiempos de diagnóstico. teniendo como base el ODS 3 (Salud yBienestar)

Objetivos del Proyecto

Objetivo General

Desarrollar una aplicación web basada en un modelo de ML "CNN (DenseNet)" para la detección automática de neumonía en radiografías de tórax, que ayude a reducir los tiempos de espera en diagnósticos y la carga en sistemas de salud con escasez de especialistas.

Objetivo Específico 1

Implementar una red neuronal convolucional (DenseNet121), optimizando métricas para minimizar falsos negativos.

Objetivo Específico 2

Evaluar el modelo con radiólogos, comparando sus resultados con diagnósticos manuales para garantizar confiabilidad..

Objetivo Específico 3

Crear una interfaz en Flask que permita a los médicos subir radiografías y recibir diagnósticos en segundos, con explicaciones interpretables (Grad-CAM).

Objetivo de Desarrollo Sostenible

De acuerdo a la justificacion anterior, cabe mencionar, que se el proyecto se realizara acorde al objetivo de desarrollo sostenible numero 3 Salud y bienestar

Evaluación del Dataset

TEl dataset de Kaggle (Chest X-Ray Images) es adecuado para el proyecto, ya que contiene 5,863 radiografías etiquetadas (neumonía/normal) validadas por expertos, con resolución suficiente (1,024x1,024 px) para detectar patrones clínicos. Aunque está limitado a pacientes pediátricos, su tamaño y calidad permiten entrenar un modelo confiable para el diagnóstico automático, cumpliendo con el objetivo clínico prioritario..
