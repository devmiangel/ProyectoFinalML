Problema

Planteamiento

La neumonía es una de las principales causas de mortalidad infantil a nivel global (OMS, 2023). Su diagnóstico temprano mediante rayos X es crítico

en regiones con escasez de radiólogos, la interpretación de imágenes puede retrasarse

Saturación de sistemas de salud en áreas rurales.
Falta de estandarización en la evaluación de imágenes.

Justificación

Este proyecto busca desarrollar un modelo de machine learning para detectar neumonía en radiografías de tórax.
Utilizando un data set público de Kaggle (Chest X-Ray Images), se entrena un modelo de ML "CNN (convolucional neuronal network)" para clasificar imágenes, facilitando la lectura de radiografías, se propone una aplicación web en flask que optimiza recursos médicos y disminuye tiempos de diagnóstico interpretados por los especialistas.

Objetivos del Proyecto

Objetivo General

Desarrollar una aplicación web basada en un modelo de ML "CNN (red neuronal convolucional )" para la detección automática de neumonía en radiografías de tórax, que ayude a reducir los tiempos de espera en diagnósticos y la carga en sistemas de salud con escasez de especialistas

Objetivo Específico 1

Implementar una red neuronal convolucional (CNN), optimizando métricas para minimizar falsos negativos

Objetivo Específico 2

Evaluar el modelo con radiólogos, comparando sus resultados con diagnósticos manuales para garantizar confiabilidad.

Objetivo Específico 3

Crear una interfaz en Flask que permita a los médicos subir radiografías y recibir diagnósticos en segundos, con explicaciones interpretables

Objetivo de Desarrollo Sostenible

De acuerdo a la justificacion anterior, cabe mencionar, que se el proyecto se realizara acorde al objetivo de desarrollo sostenible numero 3 Salud y bienestar

Evaluación del Dataset

El dataset de Kaggle (Chest X-Ray Images) es adecuado para el proyecto, ya que contiene 5,863 radiografías etiquetadas (neumonía/normal) validadas por expertos, con resolución suficiente (1,024x1,024 px) para detectar patrones clínicos. Aunque está limitado a pacientes pediátricos, su tamaño y calidad permiten entrenar un modelo confiable para el diagnóstico automático, cumpliendo con el objetivo clínico prioritario
